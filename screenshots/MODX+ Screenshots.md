# MODX+ Screenshots

## USB Drive

Use a USB drive formatted with `FAT32`.

You can format the drive on the MODX+ itself if necessary.

## Take Screenshot

* Insert the USB drive into the MODX+ USB port
* Navigate to the screen you want to capture
* Press and hold `A/D INPUT ON/OFF`
* Press and release `AUDITION`
* Release `A/D INPUT ON/OFF`

The screenshot will be saved to the USB drive, named `DSNAP_#.png`, where `#` is a sequential number.

An on-screen notification will *not* be shown when taking the screenshot.
